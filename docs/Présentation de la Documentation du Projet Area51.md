# Documentation du Projet Area51

## Vue d'ensemble

Cette documentation est organisée en plusieurs sections pour faciliter la compréhension et l'utilisation du projet Area51.

## Structure de la documentation

### 1. Documentation des endpoints
Référence complète des endpoints API de chaque service avec leurs paramètres et réponses.

### 2. Documentation du code
Explication détaillée du fonctionnement interne des services et de leur implémentation.

### 3. Technologies utilisées
Présentation de chaque technologie employée dans le projet avec leur rôle et justification.

### 4. Guide de démarrage
Instructions pas à pas pour installer et lancer le projet en local.
